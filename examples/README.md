# Examples

Real-world examples demonstrating the use of prompts and tools from this library.

## Structure

Examples are organized by scenario, showing:
- Before and after code samples
- Prompt usage and results
- Multi-step workflows
- Common patterns and solutions

## Contributing Examples

When adding examples, include:
1. **Context** - What was the goal?
2. **Prompts Used** - Which prompts/tools were applied
3. **Results** - Code samples, outputs, or explanations
4. **Lessons Learned** - What worked well, what to improve

This helps demonstrate the practical value of the library.
