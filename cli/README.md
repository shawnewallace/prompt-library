# prompt-library CLI

CLI tool to quickly install AI assistant agents and prompts from the prompt-library repository.

## Installation

```bash
npm install -g @shawnwallace/prompt-library
```

## Usage

```bash
# Initialize in a new project
prompt-library init

# List available agents and prompts
prompt-library list

# Add a specific agent or prompt
prompt-library add hemingway

# Check for updates
prompt-library check

# Update installed items
prompt-library update hemingway
```

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Lint code
npm run lint

# Format code
npm run format
```
